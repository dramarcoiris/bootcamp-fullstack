Ejercicio 5 · Página de presentación de un evento
Objetivo
Integrar metadatos, estructura semántica, listas, enlaces e información organizada en un caso
realista.

Qué tienes que construir
Crea una página de presentación para un evento. Puede ser, por ejemplo:

    meetup de desarrollo web;
    taller de introducción a React;
    hackathon;
    jornada de empleabilidad tecnológica;
    bootcamp open day.

La página debe incluir

    título del evento;
    fecha y lugar;
    descripción breve;
    agenda resumida;
    ponentes o participantes;
    información de inscripción;
    contacto;
    enlaces relevantes.

Condiciones obligatorias
Debes usar:

    header
    main
    section
    footer
    una lista ordenada o no ordenada para la agenda;
    un enlace externo;
    un enlace mailto: o tel:

Requisito adicional
Incluye un favicon enlazado en el head , aunque sea provisional.

Resultado esperado
Una base HTML muy parecida a una página real de evento informativo.
