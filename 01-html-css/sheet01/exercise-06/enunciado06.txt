Ejercicio 6 · Auditoría y reconstrucción de una estructura defectuosa

Objetivo
Detectar y corregir errores de estructura, semántica y uso de atributos.

Qué tienes que hacer
A continuación tienes un HTML deliberadamente mejorable. No debes copiarlo tal cual: debes
reconstruirlo correctamente en tu archivo.

Código de partida

<!DOCTYPE html>
<html>
<head>
<title>Mi web</title>
</head>
<body>
<div class="top">
<div><a href="https://example.com" target="_blank">haz clic aquí</a></div>
<div>Mi curso</div>
</div>
<div>
<div>Aprende HTML rapido</div>
<div>Esto es un curso intensivo para aprender mucho.</div>
<img src="curso.jpg">
</div>
<div>
<div>Temario</div>
<div>HTML</div>
<div>CSS</div>
<div>JS</div>
</div>
</body>
</html>

Tu tarea
Reescribe esa página para convertirla en una versión mejor construida.

Debes corregir, como mínimo

    idioma del documento;
    metadatos básicos;
    jerarquía de encabezados;
    semántica estructural;
    texto de enlace;
    uso de rel con target="_blank" ;
    imagen sin alt ;
    listas mal resueltas;
    organización general del contenido.
