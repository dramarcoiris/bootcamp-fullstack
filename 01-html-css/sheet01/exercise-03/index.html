Preguntas frecuentes - Bootcamp

En este documento se responden las preguntas más comunes sobre el Bootcamp Fullstack JAVA9

Índice de preguntas

  • Duración del Bootcamp
  • Conocimientos previos necesarios
  • Proyectos desarrollados durante el Bootcamp

Duración del Bootcamp

  • ¿Cuánto dura el Bootcamp?

    Este Bootcamp tiene una duración de 336 horas, comenzando el 9 de Marzo y terminando el 12 de Junio.

  • ¿Cuántas clases hay a la semana?

    Tenemos clases presenciales de lunes a viernes, cada día de 9:00 a 14:00 horas.

  • ¿Hay algún descanso durante el Bootcamp?

    Sí, hay un descanso de 30 minutos a las 10:15

  • ¿Cuándo es el examen final?

    En esta formación no hay examen final, pero sí cuenta con la entrega de un proyecto final.

Conocimientos previos necesarios

  • ¿Es necesario tener conocimientos previos de programación?

    No es necesario tener conocimientos previos de programación, aunque sí es recomendable tener una base en lógica de programación.

  • ¿Se necesita experiencia en Java para el Bootcamp?

    No se necesita experiencia previa en Java, pero es recomendable.

  • ¿Qué nivel de inglés se requiere?

    No se requiere nivel de inglés avanzado, solo el necesario para el lenguaje de programación.

  • ¿Se necesita tener ordenador para realizar el Bootcamp?

    Sí, es necesario tener un ordenador con acceso a internet para poder participar en las clases y realizar los ejercicios.

Proyectos desarrollados durante el Bootcamp

  • ¿Qué tipo de proyectos se desarrollan durante el Bootcamp?

    Se desarrollan proyectos relacionados con el desarrollo web, utilizando tecnologías como HTML, CSS, JavaScript, React y Java.

  • ¿Se trabaja en equipo para los proyectos?

    Sí, se fomenta el trabajo en equipo y se realizan proyectos grupales para simular un entorno de trabajo real.

  • ¿Se entregan proyectos individuales?

    Sí, además de los proyectos grupales, también se asignan proyectos individuales para evaluar el progreso de cada estudiante.

  • ¿Se realiza algún proyecto final?

    Sí, al finalizar el Bootcamp, los estudiantes realizan un proyecto final en grupo.
