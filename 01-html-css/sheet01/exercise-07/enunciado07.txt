Ejercicio 7 · Mini proyecto integrador de la hoja

Objetivo
Aplicar de forma conjunta casi todo lo trabajado en esta primera hoja.

Qué tienes que construir
Elige uno de estos escenarios:

1. Página informativa de una academia tecnológica.
2. Web básica de una startup ficticia.
3. Portal de presentación de una herramienta digital.
4. Página de una comunidad técnica o meetup.
5. Sitio de un servicio digital especializado.

La página debe incluir, como mínimo

    documento HTML completo;
    lang , charset , viewport y title ;
    meta description ;
    favicon;
    cabecera;
    navegación;
    contenido principal;
    varias secciones;
    al menos un article o aside si tiene sentido;
    encabezados bien jerarquizados;
    una lista;
    una imagen con alt ;
    un figure con figcaption ;
    un enlace interno;
    un enlace externo con target y rel correctos;
    una cita breve o extensa;
    un pie de página.

Requisito de calidad
Antes de dar el ejercicio por terminado, comprueba que:

    no has abusado de div ;
    el contenido se entiende por su estructura;
    los enlaces tienen textos claros;
    las imágenes tienen alt adecuado;
    la indentación es consistente;
    la página podría servir como base real de una interfaz posterior.
